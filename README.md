# Desafio Trilha POO - Smartphone

Projeto em C# desenvolvido para praticar os conceitos de Programação Orientada a Objetos (POO), utilizando uma classe abstrata `Smartphone` e as classes concretas `Nokia` e `Iphone`.

## Funcionalidades

- Criação de objetos `Nokia` e `Iphone`.
- Métodos para:
  - Ligar.
  - Receber ligação.
  - Instalar aplicativos em cada tipo de smartphone (lojas diferentes).

## Tecnologias utilizadas

- .NET SDK (ex.: .NET 6 ou superior).
- Linguagem C#.
- Editor: Visual Studio Code (ou outro de sua preferência).
- Git e GitHub para versionamento de código.
